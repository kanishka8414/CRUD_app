# 📋 CRUD Application using JavaScript

This is a simple **CRUD (Create, Read, Update, Delete)** application built using **HTML, CSS, and JavaScript**. It allows users to add, view, edit, and delete records dynamically in the browser without any backend.

## 🚀 Features

- 📝 **Create** new entries (e.g., user info, tasks, notes)
- 📖 **Read** and display the entries in a table or list
- ✏️ **Update** existing records
- ❌ **Delete** any record
- 🗃️ Data is stored temporarily in **browser memory**

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- (Optional: Bootstrap for styling)
